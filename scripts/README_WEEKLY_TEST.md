# Скрипт симуляции недельного использования приложения

Этот скрипт автоматически выполняет действия пользователя на эмуляторе, симулируя недельное использование приложения.

## Требования

1. Эмулятор Android должен быть запущен
2. Приложение должно быть установлено на эмуляторе
3. PowerShell (для Windows)

## Использование

1. Убедитесь, что эмулятор запущен и приложение открыто:
```powershell
flutter run
```

2. В другом терминале запустите скрипт:
```powershell
cd scripts
.\weekly_usage_simulation.ps1
```

## Что делает скрипт

### День 1: Изучение приложения
- Просмотр всех основных экранов (Баланс, Копилки, Календарь, Уроки, Настройки)
- Изучение интерфейса

### День 2: Работа с транзакциями
- Добавление дохода (1000)
- Добавление расхода (300)

### День 3: Работа с копилками
- Создание новой копилки "Новая игра" на 5000

### День 4: Планирование событий
- Создание события "Подарок на день рождения" на 2000

### День 5: Использование инструментов
- Открытие калькулятора бюджета
- Использование калькулятора

### День 6: Взаимодействие с Бари
- Открытие чата с Бари
- Задание вопроса "Сколько у меня денег"

### День 7: Уроки и финальная проверка
- Просмотр урока
- Проверка статистики

## Настройка координат

Если координаты не совпадают с вашим эмулятором, отредактируйте скрипт:

1. Узнайте разрешение вашего эмулятора:
```powershell
adb shell wm size
```

2. Отредактируйте координаты в скрипте под ваше разрешение

## Примечания

- Скрипт использует примерные координаты для эмулятора 1080x1920
- Может потребоваться настройка координат под ваш эмулятор
- Все действия выполняются с задержками для стабильности
